Learning Context:
Outcome Filter

Name the student thinking and related assessment first, then choose interactive media only when it helps learners practice or demonstrate that thinking.

01

The Filter

Move from the learning outcome to the smallest interaction that supports the needed thinking.

StepAskWrite
1. OutcomeWhat should students be able to do?Students will be able to...
2. EvidenceWhat would show progress toward the outcome?A comparison, explanation, prediction, calculation, classification, interpretation...
3. Cognitive actionWhat thinking does that evidence require?Notice, compare, predict, calculate, sequence, explain, classify, interpret...
4. Useful changeWhat should change on screen?Input, time, state, comparison, sequence, consequence, or feedback
5. InteractionWhich small form supports that change?Simulation, case, diagram, scenario, practice exercise, reflection...
02

The Change Test

If nothing changes, you probably do not need interactive media. A Canvas page, diagram, short video, or worked example may be enough.

Interactive media is more useful when students need to:

  • See a process unfold over time.
  • Try a choice and observe a result.
  • Compare two or more cases.
  • Adjust one variable and inspect the effect.
  • Reveal layers of an image, graph, source, or scenario.
  • Practice a decision before higher-stakes work.
03

Quick Example

Use the five steps to make a small, purposeful interaction choice.

Outcome

Students will predict how changing a dosage affects concentration over time.

Evidence

An explanation connecting dosage to the changing concentration curve.

Cognitive action

Predict a relationship and interpret a trend.

Useful change

One variable changes and the graph updates.

Interaction

Single-variable model with a slider or preset choices.

Decision rule

Build the smallest artifact that lets students practice the target thinking more clearly than static content would.

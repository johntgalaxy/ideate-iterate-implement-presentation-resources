The Map Becomes
a First-Build Brief

Copy this structure and replace the bracketed notes with the decisions you recorded on page one.

Create a short, accessible learning activity for [course or module].

Learning Context
The activity supports this outcome: [student outcome]. Learners need to [practice or understand]. Use it as [prepare, practice, review, or enrichment] after [what comes before] and before [what comes after].

Interaction Design
Use a [small interaction pattern]. Learners will [describe the learner action] and see [what changes or feedback appears]. Include [brief explanation, reflection, or transfer move].

Expert Content
Accurately use [approved facts, terms, sources, data, examples, and assets]. Address [misconception or boundary]. Do not invent, replace, or extend [content that requires instructor verification].

Artifact Specification
Create a [length and format] artifact with [controls and instructions]. It must be [accessibility and responsive requirements]. Do not use [prohibited features]. Prepare it for [Canvas placement and/or SCORM requirement].

Keep the code self-contained, reviewable, and easy to revise.

ITERATE after the first build

Diagnose through one lens.

Misses the point?
Review Learning Context.
Feels like a demo?
Review Interaction Design.
Gets facts wrong?
Review Expert Content.
Fails visually or technically?
Review Artifact Specification.
